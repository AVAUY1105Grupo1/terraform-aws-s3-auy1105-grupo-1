# Ejemplos de uso del Módulo de Almacenamiento

## Objetivo del ejemplo
Esta carpeta provee una guía práctica para instanciar el módulo de S3 de forma aislada. Demuestra cómo pasar las variables para crear un bucket con acceso público habilitado, ideal para almacenar assets web (como imágenes o archivos estáticos).

## Instrucciones de ejecución
1. Ingresa a la carpeta del ejemplo:
   ```bash
   cd examples/despliegue_bucket
2. Ejecuta los comandos habituales para descargar proveedores y crear la infraestructura:
 ```
terraform init
terraform plan
terraform apply
```
